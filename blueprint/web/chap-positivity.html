6 Positivity and sum-of-squares

Density bounds are inequalities in a semantic order on \(\mathcal{A}^\sigma \) induced by its positive homomorphisms. Sum-of-squares certificates are the mechanism that proves them. (Source: FlagAlgebra/PositiveHom.lean, QuadraticForm.lean.)

Definition 61 Positive homomorphism
✓
#

Uses

  • Definition 31
  • Definition 69

Lean declarations

  • FlagAlgebras.PositiveHom

A positive homomorphism is an \(\mathbb {R}\)-algebra homomorphism \(\varphi : \mathcal{A}^\sigma \to \mathbb {R}\) that is nonnegative on every flag. These are the semantic evaluations of the algebra — the limits of subgraph densities.

Definition 62 Semantic cone
✓
#

Uses

Lean declarations

The semantic cone is \(\{ \, f \mid \varphi (f) \ge 0 \text{ for all positive } \varphi \, \} \). Membership is exactly an unconditional density inequality.

Definition 63 Semantic order
✓
#

Uses

  • Definition 62

Lean declarations

  • FlagAlgebras.le_def

Define \(f \le g\) to mean \(g - f\) lies in the semantic cone, i.e. every positive homomorphism satisfies \(\varphi (f) \le \varphi (g)\). This is a preorder on \(\mathcal{A}^\sigma \), the order in which all bounds are stated.

Definition 66 Flag quadratic form
✓

Given a matrix \(M\) and a vector \(v = (v_1,\dots ,v_n)\) of flag-algebra elements,

\[ \mathrm{flagQuadraticForm}\, M\, v \; =\; \sum _{i,j} M_{ij}\, (v_i\, v_j), \]

the raw sum-of-squares certificate object.

Theorem 67 PSD forms are nonnegative
✓
#

Uses

  • Definition 66
  • Definition 62
  • Definition 63

Lean declarations

  • FlagAlgebras.flagQuadraticForm_nonneg

If \(M \succeq 0\) (positive semidefinite) then \(\mathrm{flagQuadraticForm}\, M\, v \ge 0\) in the semantic cone.

Proof ▶

Apply any positive homomorphism \(\varphi \): it turns the form into \(\sum _{ij} M_{ij}\, \varphi (v_i)\varphi (v_j) = x^\top M x \ge 0\) with \(x_i = \varphi (v_i)\), since \(M \succeq 0\).

Theorem 68 Unlabeled PSD forms are nonnegative
✓
#

Uses

  • Theorem 67

Lean declarations

  • FlagAlgebras.flagQuadraticForm_downward_nonneg

The unlabeled form \([\! [\, \mathrm{flagQuadraticForm}\, M\, v\, ]\! ]_0 \ge 0\) for \(M \succeq 0\). This is the certificate actually used, since density bounds live in the unlabeled algebra \(\mathcal{A}^\emptyset \).

Proof ▶

The form is in the semantic cone by Theorem 67, and downward preserves the semantic cone (Theorem 84), so its unlabeling is nonnegative.
