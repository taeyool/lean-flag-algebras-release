8 The computational layer

To actually run flag-algebra computations (enumerate flags, compute densities) at elaboration time, the abstract objects are mirrored by executable ones: graphs encoded as explicit edge sets \(\mathrm{Finset}(\mathrm{Sym}_2(\mathrm{Fin}\, n))\). Every computable definition is paired with a theorem proving it equals its abstract counterpart, so computed results are provably correct. (Source: FlagAlgebra/Compute/; all names in the FlagAlgebras.Compute namespace.)
